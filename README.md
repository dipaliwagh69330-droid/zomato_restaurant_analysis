# Zomato Restaurant Analysis
- This is an end to end data analysis project based on Zomato restaurant data.
In this project I worked on data cleaning, exploratory data analysis, SQL queries and an interactive Power BI dashboard.
- The main goal of this project was to understand restaurant trends, customer ratings, pricing patterns and ordering behaviour using real world style data.

**Tools Used**
- Python
- Pandas
- Matplotlib
- MySQL
- Power BI
  
**Things I Worked On**
- Cleaned and prepared raw restaurant data using Python
- Handled missing values and formatting issues
- Performed EDA to find patterns and business insights
- Wrote SQL queries for filtering, grouping and aggregation
- Built an interactive Power BI dashboard with KPI cards and slicers
  
**Some Insights Found**
- Online ordering restaurants had slightly better average ratings
- Restaurants with table booking generally showed higher ratings
- Few restaurants had very high customer engagement compared to others
- Some areas contained higher average restaurant costs
- Most ratings were concentrated around 3.5 to 4.2 range
  
**Dashboard Features**
- KPI Cards
- Top restaurant areas
- Rating distribution
- Online order vs ratings
- Table booking vs ratings
- Average cost analysis
- Interactive slicers and filtering
